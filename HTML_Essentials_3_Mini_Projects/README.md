# HTML Essentials — 3 Mini Projects (Mentor + Mentee)

Inside this folder you have 3 different mini projects.  
Each project contains:
- **MENTEE/** (instructional comments only)
- **MENTOR/** (same instructional comments + finished code)
- A project-level **README.md** explaining what you’ll learn

## Projects
1. **SEO Profile Card**
2. **Recipe Planner: Table + Lists**
3. **Community Event Signup**

## How to use
1. Pick a project folder.
2. Start in **MENTEE/** and follow the comments step-by-step.
3. When you finish (or get stuck), compare with **MENTOR/**.
4. Open `index.html` in your browser to see your result.
